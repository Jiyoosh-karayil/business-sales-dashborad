# Power BI Analytics Dashboard

A comprehensive business intelligence dashboard created with Microsoft Power BI, featuring interactive visualizations, custom visuals, and data-driven insights.

![Power BI](https://img.shields.io/badge/Power_BI-F2C811?style=flat&logo=powerbi&logoColor=black)
![DAX](https://img.shields.io/badge/DAX-FF6F00?style=flat&logo=microsoft&logoColor=white)

## Overview

This Power BI project demonstrates advanced data analysis and visualization techniques, including custom visuals, interactive dashboards, and business intelligence reporting capabilities.

## Features

- **Interactive Dashboards** - Dynamic filtering and cross-highlighting across all visuals
- **Custom Visuals** - Advanced visualizations including:
  - Advanced Gauge charts
  - Calendar Visual for time-based analysis
  - Histogram charts for distribution analysis
  - Timeline visualizations
  - Text filters for enhanced data exploration
  - Stacked Insights BI visuals
- **Data Analysis** - Comprehensive metrics and KPIs
- **Responsive Design** - Optimized for various screen sizes
- **Professional Theme** - Custom icons and branding elements

## Technologies Used

- **Microsoft Power BI Desktop** - Primary development tool
- **DAX (Data Analysis Expressions)** - Advanced calculations and measures
- **Power Query** - Data transformation and preparation
- **Custom Visuals** - Enhanced visualization capabilities from AppSource

## Custom Visuals Included

1. **Advanced Gauge** - Enhanced gauge visualizations
2. **Calendar Visual** - Time-based data display
3. **Histogram** - Distribution analysis
4. **Histogram XY** - Advanced distribution charts
5. **Stacked Insights BI** - Layered data insights
6. **Text Filter** - Dynamic text-based filtering
7. **Timeline** - Chronological data visualization
